## 📁 项目文件结构

### 源码文件（16 个 Scala 文件）

| 文件 | 行数（约） | 说明 |
|------|-----------|------|
| `Token.scala` | — | Token 定义 + Position + PositionedToken |
| `Lexer.scala` | — | 词法分析器（含注释支持、行号/列号跟踪） |
| `AST.scala` | — | AST 节点定义（所有语句/表达式/列/表引用类型） |
| `Parser.scala` | — | 递归下降解析器（含位置感知错误报告） |
| `ParseException.scala` | — | 带位置的解析异常 + 源码上下文 + `^` 指示器 |
| `ASTVisitor.scala` | — | Visitor 模式框架（遍历/变换/PrettyPrinter/提取器） |
| `SemanticAnalyzer.scala` | — | 传统语义分析 |
| `SemanticVisitor.scala` | — | Visitor 管道语义分析 |
| `TypeChecker.scala` | — | 类型系统（SQLType/TypeInferencer/TypeChecker/TypeCheckVisitor） |
| `ASTJsonSerializer.scala` | — | AST → JSON 序列化器 |
| `ExecutionEngine.scala` | ~1146 | 内存数据库 + 表达式求值 + SQL 执行器 |
| `QueryPlan.scala` | ~639 | 逻辑执行计划 + 优化器 + 可视化 |
| `MySQLParser.scala` | — | 统一入口（parse/format/toJson/analyze/explain） |
| `SQLRepl.scala` | — | 交互式 REPL 命令行 |
